# 智膳康 - 膳食营养健康系统

## 项目简介

智膳康是一个专业的膳食营养健康管理系统，致力于为用户提供科学、个性化的营养建议和健康管理方案。通过智能分析用户的膳食结构和健康数据，帮助用户建立科学的膳食习惯，提高健康水平，享受高品质的生活。

## 功能特性

### 核心功能
- **营养分析**：智能分析膳食结构，提供详细的营养成分报告和健康建议
- **膳食计划**：根据用户身体状况和需求，定制个性化的膳食计划
- **健康追踪**：实时追踪健康数据，包括体重、体脂、BMI等
- **专家咨询**：在线咨询专业营养师，获取权威的健康指导
- **食谱推荐**：提供多样化的健康食谱，满足不同口味和需求
- **运动建议**：结合膳食和运动，提供全面的健康管理方案

### 个人中心
- 个人信息管理：修改头像、个人简介等
- 营养摄入跟踪：记录每日饮食，查看营养成分摄入情况
- 身体数据记录：添加和查看体重、体脂率等身体指标
- 膳食计划管理：查看和继续已保存的膳食计划
- 健康目标设定：设定和追踪健康目标

## 技术栈

- **前端**：HTML5, CSS3
- **图标库**：Font Awesome
- **响应式设计**：适配不同屏幕尺寸
- **本地开发服务器**：Python HTTP Server

## 使用说明

### 首页功能
1. 浏览系统核心功能介绍
2. 了解系统的主要特点和优势
3. 通过导航栏访问个人中心和其他页面

### 个人中心功能
1. **营养跟踪**：查看每日营养摄入情况，点击"记录饮食"添加新的饮食记录
2. **身体数据**：查看身体指标变化，点击"添加测量"记录新的身体数据
3. **已保存计划**：查看和管理已保存的膳食计划
4. **健康目标**：查看当前健康目标的完成进度

## 设计特点

- **绿色主题**：采用绿色作为主色调，象征健康、自然和活力
- **响应式设计**：适配桌面端、平板和手机等不同设备
- **现代化UI**：采用卡片式布局、渐变效果和动画过渡，提供良好的视觉体验
- **直观的数据展示**：通过图表、进度条等方式直观展示健康数据
- **用户友好**：清晰的导航结构，易于操作和使用
