# 家庭食谱与膳食规划应用 - 项目总结

## 🎯 项目概述

本项目是一个完整的桌面应用程序，使用Python+Flask+PyQt6+SQLite技术栈开发，专为家庭用户设计的食谱管理与膳食规划工具。

## ✅ 已完成的核心功能

### 1. 用户认证系统
- 用户注册/登录功能
- JWT令牌认证机制
- 用户数据隔离

### 2. 食谱管理模块
- 食谱的增删改查功能
- 食谱分类管理（早餐、午餐、晚餐、甜点等）
- 难度评级系统
- 食材清单和制作步骤管理
- 食谱搜索功能

### 3. 膳食规划模块
- 按日期的膳食计划制定
- 多餐次支持（早餐、午餐、晚餐、加餐）
- 智能膳食计划生成
- 食谱与膳食计划关联

### 4. 食材管理模块
- 食材库存管理
- 食材分类（蔬菜、水果、肉类、海鲜等）
- 保质期提醒功能
- 库存数量管理

### 5. 购物清单模块
- 从膳食计划自动生成购物清单
- 手动添加购物项
- 购物清单导出功能

### 6. 营养分析模块
- 每日营养摄入统计
- 营养成分分析（蛋白质、脂肪、碳水化合物等）
- 营养数据可视化图表
- 健康建议生成

### 7. 数据可视化
- 营养分布饼图
- 卡路里趋势图
- 营养素对比柱状图
- 三餐分析图

## 📁 项目文件结构

```
项目目录/
├── simple_app.py              # Flask后端主程序
├── main_window.py            # PyQt6主窗口应用
├── nutrition_charts.py       # 营养数据可视化模块
├── start_app.py              # 一键启动脚本
├── test_app.py               # 功能测试脚本
├── requirements.txt          # Python依赖包
├── README.md                 # 项目说明文档
└── database/                 # 数据库文件目录
    └── recipe_app.db        # SQLite数据库文件
```

## 🚀 快速启动

### 1. 安装依赖
```bash
pip install flask flask-cors flask-jwt-extended sqlalchemy requests pyqt6 matplotlib
```

### 2. 启动后端
```bash
python simple_app.py
```

### 3. 启动前端
```bash
python main_window.py
```

或使用一键启动：
```bash
python start_app.py
```

### 4. 功能测试
```bash
python test_app.py
```
